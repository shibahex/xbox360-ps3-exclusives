## Repository Structure

The core dataset is organized into simple text databases:
* **[`xbox360 games`](./xbox360_games.txt)**: A list of Xbox 360 game names scraped from Wikipedia.
* **[`ps3 games`](./ps3_games.txt)**: A list of PS3 game names scraped from Wikipedia.
* **[`xbox360 exclusives`](./xbox360_exclusives.txt)**: A compiled checklist of games unique to the Xbox 360 system (including Kinect-only titles).
* **[`ps3 exclusives`](./ps3_exclusives.txt)**: A compiled checklist of games unique to the PlayStation 3 system (including ps3 versions of games).


## Goal

The purpose of this repository is to:
- Preserve information for Xbox 360 and PS3 software.
- Provide lists for archival, scripting, and research purposes.
- Help compare platform libraries and discover forgotten exclusives.

## Data Collection

Game lists were scraped from publicly available Wikipedia software title pages and then compared against each other to determine exclusivity.
Because naming inconsistencies, regional variants, and alternate spellings exist, some titles may require manual verification.

This repository is intended for educational, archival, and research purposes.
